# Planes — marketing & legal site

Next.js 14 (App Router) site served at **planesapp.xyz**.

Routes:
- `/` — home
- `/license` — end-user licence agreement
- `/copyright` — copyright + DMCA
- `/privacy` — privacy policy

All contact is consolidated at `digitalpenguininc@gmail.com`.

## Develop

```
npm install
npm run dev
```

Visit http://localhost:3000.

## Deploy

Designed to run on Vercel. Import the GitHub repo in the Vercel dashboard,
add `planesapp.xyz` as the production domain, done — no env vars required.
